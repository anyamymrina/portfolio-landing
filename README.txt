Лендинг (статический сайт)

Как запустить локально:
1) Откройте папку и дважды кликните index.html
или
2) Если хотите локальный сервер: установите расширение Live Server (VS Code) и запустите.

Как задеплоить и получить ссылку:
Самый простой путь: GitHub Pages.
Создайте репозиторий, залейте содержимое папки, включите Pages для ветки main.
Либо Netlify: перетащите папку в Netlify (Drop), получите URL.

Что заменить:
- Имя/контакты в index.html
- Ссылки на Behance/видео/PDF
- Превью-картинки в assets (файлы thumb-*.png и портрет)
